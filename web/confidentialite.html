Politique de confidentialité & RGPD

justicelibre.org est un service citoyen gratuit d'accès à la jurisprudence publique. Nous collectons le strict minimum de données et ne commercialisons rien.

Ce qu'on NE collecte PAS

  • Pas de cookies de tracking, pas de cookies publicitaires. Aucun consentement de ce type n'est requis.
  • Pas d'analytics (Google Analytics, Matomo, etc.).
  • Pas de logs d'IP : les adresses IP ne sont pas enregistrées de manière persistante lors de vos recherches ou de l'utilisation du MCP. Seules les logs techniques nginx conservent les IP pendant 7 jours (protection contre abus), après quoi elles sont purgées.
  • Pas de compte utilisateur. Aucune inscription, aucun profil, aucune session utilisateur.
  • Pas de contenu de requêtes historisé. Vos recherches ne sont pas stockées.

Ce qu'on utilise (côté navigateur uniquement)

  • localStorage : stockage local (non envoyé au serveur) pour :
    • Préférence de thème (clair/sombre)
    • Cache court-terme (24h) des articles de loi consultés, pour éviter les allers-retours inutiles
    Ces données ne quittent jamais votre appareil. Vous pouvez les effacer à tout moment via les outils de votre navigateur.

Ce qu'on collecte (uniquement sur action volontaire de votre part)

1. Token PISTE (échange OAuth2)

Si vous utilisez le tutoriel PISTE pour générer un token de session, vos identifiants Client ID et Client Secret PISTE sont :

FinalitéÉchanger ces identifiants contre un Bearer token PISTE et vous retourner un jeton de session temporaire
Base légaleExécution du service demandé par vous (art. 6.1.b RGPD)
ConservationLes identifiants ne sont JAMAIS stockés sur disque. Ils transitent en RAM le temps de l'échange avec PISTE, puis sont effacés.
Bearer PISTE obtenuConservé 1 heure maximum, associé à un jeton de session justicelibre anonyme (UUID), puis supprimé automatiquement

2. Formulaire de signalement (bouton « Problème ? »)

Si vous utilisez le formulaire de signalement d'erreur dans le panneau article :

Données collectéesVotre message (obligatoire), votre email (optionnel, seulement si vous voulez une réponse), et un identifiant navigateur tronqué (User-Agent, pour débug)
Ce qui N'EST PAS collectéVotre IP n'est PAS enregistrée
FinalitéAméliorer la détection automatique des citations d'articles de loi
Base légaleIntérêt légitime (amélioration du service public d'accès au droit, art. 6.1.f RGPD)
Conservation12 mois maximum. Purge automatique mensuelle au-delà.
DestinatairesUniquement le mainteneur du site. Aucune transmission à des tiers.

Vos droits RGPD

Conformément au Règlement général sur la protection des données (art. 15 à 22), vous disposez des droits suivants :

  • Droit d'accès : obtenir la confirmation que des données vous concernant sont traitées, et une copie de ces données
  • Droit de rectification : corriger des données inexactes
  • Droit à l'effacement : demander la suppression de vos signalements (nous réécrivons le fichier en excluant vos entrées identifiables par votre email)
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit à la limitation : restreindre le traitement
  • Droit à la portabilité : recevoir vos données dans un format structuré

Pour exercer ces droits, écrivez à contact@justicelibre.org. Réponse sous 1 mois maximum.

En cas de désaccord, vous pouvez saisir la CNIL à l'adresse cnil.fr.

Sous-traitants & transferts de données

  • Hetzner Online GmbH (Allemagne, UE) — hébergeur. Données stockées en UE.
  • Cloudflare (le cas échéant, en front CDN) — proxy réseau. Peut voir les IP de connexion (loggées brièvement par Cloudflare selon leurs propres politiques).
  • DILA, Conseil d'État, Cour de cassation, HUDOC, EUR-Lex : sources publiques interrogées pour fournir le service. Aucune de vos données personnelles ne leur est transmise.

Aucun transfert de données hors UE.
